# ✅ Todo App

A clean and responsive Todo App that helps users manage daily tasks efficiently. Users can add, edit, complete, and delete tasks through a simple and user-friendly interface, making task management easy and organized.

## ✨ Features
- Add New Tasks
- Edit Existing Tasks
- Mark Tasks as Completed
- Delete Tasks
- Responsive Design
- Clean & Modern User Interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Git
- GitHub
- Vercel

## 🎯 Purpose
This project demonstrates front-end development skills by building an interactive and responsive task management application using modern web technologies and JavaScript DOM manipulation.

## 🚀 Future Enhancements

- Local Storage Support
